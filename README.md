**Contribution -**
1. Clone the repo in your machine, `git clone <repo url>`
2. `cd PSI-PJP` to jump into the repo directory
3. Create a branch with name as your first name and checkout to that branch `git checkout -b <firstname>`
4. Add a text file to the local repo, write your first name in the file.
5. `git add .` to stage the file.
6. `git commit -m "Initial Commit"` to commit the changes.
7. `git push origin <firstname>` to push the changes to your branch whose name is same as your firstname.
